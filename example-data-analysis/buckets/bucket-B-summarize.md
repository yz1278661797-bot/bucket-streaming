# Bucket B: Summarize

## Summarization Strategy

1. Load search results from Bucket A
2. Identify patterns, outliers, and trends
3. Extract actionable insights

## Insight Extraction Rules

- Group related data points into themes
- Rank by significance (volume, velocity, variance)
- Flag anomalies for user attention

## Output Format

```
📋 Summary

Theme 1: [theme name]
  - Key finding: [insight]
  - Supporting data: [points]
  - Confidence: [high/medium/low]

Theme 2: ...

Anomalies detected: [count]
  - [anomaly description]

→ Handoff: chart
```
